# Skein

Skein is a local strand graph for agents and humans. It gives you a durable SQLite-backed graph of work, notes, dependencies, and workflow state, while keeping the command-line surface small and machine-readable.

## Start here

- **User reference**: the main [Skein guide](docs/skein.md) covers the model, CLI, daemon, REPL, and workspace conventions.
- **Getting started**: [install and run Skein](docs/getting-started.md) in a local workspace.
- **Spools**: [spool contracts](spools/README.md) document the shipped and reference workflow extensions, with API pages alongside each contract.
- **Specs**: [devflow tenets](devflow/TENETS.md), [philosophy](devflow/PHILOSOPHY.md), and root specs describe the source contracts behind the alpha surface.
- **Authoring**: [writing shared spools](docs/writing-shared-spools.md) and [library authoring](docs/library-authoring.md) explain how to extend Skein safely.
